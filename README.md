# 🌿 NECTAR — Cinematic Functional Beverage Experience

NECTAR is a luxury functional beverage prototype featuring butter-smooth interactions, GPU-accelerated visuals, and AI-driven sensory storytelling.

## 🎨 Brand Specification
- **Color Palette**: Nectar Teal (#1DCD9F), Obsidian Black (#000000), Champagne Cream (#FFFDD0).
- **Interactive Layer**: Universal typographic teal glow on hover for all text, titles, and buttons.
- **Performance**: 60fps transitions powered by hardware acceleration (`gpu-smooth`).
- **Design Philosophy**: Strict transparency architecture—no background boxes in the catalog or profile.

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router, Server Actions)
- **Styling**: Tailwind CSS + ShadCN UI
- **Backend**: Firebase (Auth & Firestore)
- **AI Engine**: Genkit v1.x (Google Gemini 2.5 Flash)
