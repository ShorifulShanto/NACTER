# 🌿 NECTAR — Cinematic Functional Beverage Experience

NECTAR is a luxury functional beverage prototype featuring butter-smooth interactions, GPU-accelerated visuals, and Gemini-driven sensory storytelling.

## 🎨 Brand Specification
- **Color Palette**: 
    - **Nectar Teal**: `#1DCD9F` (Primary Glow & Interaction)
    - **Obsidian Black**: `#000000` (Universal Background)
    - **Silver Dark**: `#0A0A0A` (Section Surfaces)
- **Interactive Layer**: 
    - **Universal Glow**: Silky-smooth teal text and element glow on all interactive typography and controls (Hero PREV/NEXT, Account link, Buttons).
    - **Border-Style Counters**: Hero flavor indexes (01-07) feature transparent outlines that fill with color and radiant glow on hover.
- **Performance**: 60fps cinematic transitions powered by full hardware acceleration (`gpu-smooth`).
- **Transparency Architecture**: No background boxes in the catalog, detail pages, or profile dashboard. Pure image-on-black aesthetic.

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router, Turbopack)
- **Styling**: Tailwind CSS + Framer-inspired GPU Transitions
- **Backend**: Firebase (Auth & Firestore)
- **Storage**: Cloudinary (Product Image Management)
- **AI Engine**: Genkit v1.x (Google Gemini 2.5 Flash for sensory narratives)
